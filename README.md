# Threads Client Center

Telegram-бот для работы с клиентами по Threads.

## Что уже умеет

- отдельное админ-меню;
- добавление клиентов;
- персональная ссылка для подключения клиента;
- загрузка веток одним сообщением;
- отправка веток клиенту;
- отметка «Опубликовано»;
- ежедневное внесение аналитики по каждой ветке;
- недельный отчёт;
- опрос клиента о переходах, обращениях и продажах раз в 2 дня;
- автоматическая отправка веток по расписанию.

## Переменные Railway

Добавьте в Variables:

- `BOT_TOKEN` — токен от BotFather;
- `ADMIN_ID` — ваш Telegram ID;
- `TIMEZONE` — например `Europe/Berlin`;
- `DAILY_SEND_HOUR` — час автоматической отправки веток, по умолчанию `9`;
- `RESULTS_POLL_HOUR` — час опроса клиента, по умолчанию `19`.
- `DB_PATH` — путь к базе, для Railway Volume укажите `/data/bot.db`.

## Запуск на Railway

1. Создайте новый проект.
2. Загрузите все файлы из этого архива в GitHub-репозиторий.
3. Подключите репозиторий к Railway.
4. Добавьте Railway Volume с точкой монтирования `/data`.
5. Добавьте переменные окружения, включая `DB_PATH=/data/bot.db`.
6. Нажмите Deploy.

Бот работает через long polling, отдельный домен не нужен.

## Как пользоваться

1. Зайдите в бот со своего админ-аккаунта.
2. Нажмите `👥 Клиенты`.
3. Добавьте клиента.
4. Отправьте клиенту персональную ссылку.
5. Через `📝 Ветки` выберите клиента, дату и вставьте тексты.
6. Через `📊 Внести аналитику` заносите просмотры, лайки, комментарии, репосты и изменение подписчиков.
7. Через `📈 Отчёты` смотрите сводку за 7 дней.

## Формат загрузки веток

```text
09:00
Текст первой ветки

12:00
Текст второй ветки

15:00
Текст третьей ветки

18:00
Текст четвёртой ветки
```


## Чат с темами

1. Создайте закрытую супергруппу Telegram и включите темы.
2. Добавьте бота администратором с правами отправки сообщений и управления темами.
3. Напишите в группе `/chatid`.
4. Добавьте полученный ID в Railway:
   `WORK_GROUP_ID=-100...`
5. После деплоя для новых клиентов темы создаются автоматически.
6. Сообщения клиента боту попадают в его тему.
7. Любое обычное сообщение в теме отправляется клиенту.
